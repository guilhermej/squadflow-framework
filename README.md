# SquadFlow Framework

> An open framework for administering companies that balances continuous execution and temporary initiatives.

**Status:** v1.0 in progress. First public release targeted for **2026-07-19**.

SquadFlow is a company-administration framework built around **9 first-class entities in 3 layers**:

- **Context** — Company, Brand
- **Strategy** — OKR
- **Execution** — Factory, Project, Squad, Task
- **People & Knowledge** — Player, Document

**Core insight:** *Factories run your business. Projects change it.*
